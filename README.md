# Поваренная книга А. Макаровой 1880 года
Это моя попытка не просто оцифровать одну из дореволюционных поваренных книг, но и сделать текст чуть более удобным, с возможностью поиска и ссылками внутри текста. ~~(А заодно проба пера в LaTeX'е).~~

*Планируется:*

- [ ] алфавитный указатель,
- [X] оглавление,
- [ ] иллюстрации,
- [ ] оригинальная (имперская) и современная (Си + стаканы-ложки) система мер и весов.
